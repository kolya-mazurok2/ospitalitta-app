01

House mark · два обличчя · зафіксовано

Одне коло, два обличчя. Ліва половина гладко гріє (гладкий gold-півдиск), права рве гострими язиками (ember зірка-спалах). Силует завжди круглий, гострота живе в розрізі. По центру порожнеча: половини навіть не торкаються, 8/200 прозорого зазору чистого фону, ніколи не лінія і не градієнт. Сенс знака саме в цій порожнечі.

02

Архітектура токенів · 3 рівні (DEC-016)

Компоненти споживають ЛИШЕ рівень 2 (semantic). Рівень 1 ніколи не торкається UI напряму. Тема (life / burn) перемикає тільки semantic-аліаси через [data-theme], не примітиви. Джерело істини: tokens/globals.css (Tailwind v4, CSS-first).

1 · PRIMITIVES
Сирі значення. Уся палітра, raw px.
--p-gold: #E0992E;
--p-char: #15110E;
--p-molten: #E8801F;
2 · SEMANTIC / ALIAS
Роль, не значення. Перемикається темою. Єдине, що бачать компоненти.
--brand: var(--p-gold);
--surface: var(--p-cream);
--ink: var(--p-ink);
3 · COMPONENT
Локальні до компонента, тягнуть з semantic.
--btn-bg: var(--brand);
--btn-fg: var(--on-brand);
--card-bg: var(--surface-2);
#E0992E→--brand→--btn-bgодне значення крізь 3 рівні

06

Моушн · ліво м'яко / право різко

Рух теж контрастний. Ліва половина (gold, гріє) поводиться м'яко: повільний плавний ease, дрейф, дихання. Права (ember, палить) супер різка на противагу: snap, steps, hard cut. Без glow-кросфейдів.

gold · soft

easeOutCubic, opacity 0→1, scale .82→1, дрейф translateY 26px→0. ~820ms. Дихає.

ember · snap

stepped opacity, translateX(-30px)→0 різко, hard cut. ~170ms. Без пом'якшення.

--motion-fast 120ms --motion-base 200ms --ease cubic-bezier(0.2, 0, 0, 1)
Nav logo build · on load ~1050ms

Gold-півдиск піднімається (easeOut, opacity + scale) → ember-зірка snap з translateX → wordmark витирається через clip-path inset(0 100% 0 0)→inset(0). Референс: Screen 01 Logo Motion.dc.html. Поважати prefers-reduced-motion.
